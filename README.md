# Lab 1 — Класи, об’єкти

## Опис
Це консольний застосунок на C#, який дозволяє працювати зі списком книг.  
Передбачено додавання, перегляд, пошук, зміну характеристик та видалення книг.  
Кожна книга має такі характеристики:
- Назва (`title`)
- Автор (`author`)
- Дата створення (`creationDate`)
- Кількість сторінок (`countOfPages`)
- Ціна (`price`)
- Кількість примірників (`quantity`)
- Жанр (`genre`)

## Можливості
- Встановлення максимальної кількості книг у програмі.
- Додавання нової книги з введенням усіх параметрів.
- Перегляд усіх книг у табличному вигляді.
- Пошук книги за:
  - назвою
  - автором
- Видалення книги:
  - за порядковим номером
  - за назвою
- Зміна параметрів книги:
  - зміна ціни
  - збільшення/зменшення кількості
  - порівняння дат виходу двох книг

## Структура проекту
- **Book.cs** — клас `Book` з властивостями та методами.
- **Genre.cs** — перелік жанрів (`enum Genre`).
- **Program.cs** — головна логіка програми з меню.
